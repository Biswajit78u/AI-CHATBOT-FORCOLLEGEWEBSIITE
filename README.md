# 🤖 AI Chatbot for College Website

An intelligent AI-powered chatbot for college websites to help students get instant answers about courses, admissions, events, and campus facilities.

## ✨ Features

- Real-time AI conversations
- College-specific responses  
- Mobile responsive design
- Fast and lightweight
- Easy to customize

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript
- Python Backend
- AI Model API Integration

## 🚀 Quick Start

### 1. Clone the repo
```bash
git clone https://github.com/Biswajit78u/AI-CHATBOT-FORCOLLEGEWEBSIITE.git
cd AI-CHATBOT-FORCOLLEGEWEBSIITE
```

### 2. Install backend dependencies
```bash
cd Backend
pip install -r requirements.txt
```

### 3. Setup environment variables
Create a `.env` file in Backend folder:
```env
API_KEY=your_api_key_here
PORT=5000
```

### 4. Run the app
```bash
# Start backend
python app.py

# Open frontend (new terminal)
cd ../frontend
open index.html
```

## 🔧 Environment Variables

Create `.env` file with:
- `API_KEY` - Your AI model API key
- `PORT` - Server port (default: 5000)

⚠️ Never share your `.env` file or commit it to GitHub!

## 🎯 Usage Examples

Ask questions like:
