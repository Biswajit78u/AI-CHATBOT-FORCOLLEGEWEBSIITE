# AI-CHATBOT-FORCOLLEGEWEBSIITE
# 🤖 AI Chatbot for College Website

An intelligent AI-powered chatbot designed specifically for college websites to help students, faculty, and visitors get instant answers to their queries about courses, admissions, events, campus facilities, and more.

## ✨ Features

- 💬 **Real-time AI Conversations** - Natural language processing powered by advanced AI models
- 🎓 **College-Specific Knowledge** - Answers questions about courses, fees, deadlines, and campus life
- 📱 **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- ⚡ **Fast & Lightweight** - Optimized for quick responses and smooth user experience
- 🔒 **Secure API Integration** - Environment variables protect sensitive keys
- 🎨 **Customizable UI** - Easy to match your college's branding and colors

## 🛠️ Tech Stack

**Frontend:**
- HTML5
- CSS3 (31.7%)
- JavaScript (45.1%)

**Backend:**
- Python (19.0%)
- Flask/FastAPI (AI model integration)

**Other:**
- RESTful API architecture
- Environment variables for security

## 📋 Prerequisites

Before you begin, ensure you have the following installed:

- Git
